## 本地运行 / 开发
### 技术栈
本项目是一个纯web项目，使用react+vite+tailwind+typescript开发。

### 运行
clone本项目后
```
pnpm install
```
来安装依赖
```
pnpm dev
```
来启动项目。
默认端口是 http://localhost:5173/
```
pnpm build
```
来打包项目
